# Content Rules

Do not expose identifiers, repository paths, Canonical Records, Knowledge-State terminology, validation output, hidden canon, future events, unobserved motives, or objective conclusions unavailable to the character.

Regenerate whenever a canonical source changes. If metadata is stale, sources are missing, or the prose conflicts with canon, ignore the artifact and derive a safe briefing from current canonical state.
